# BFS: результаты

## Общая информация

Для анализа производительности использовался кубический граф размером [500, 500, 500] вершин.
Для уменьшения влияния случайности на результаты измерений, время усреднялось по 5 запускам.
Последовательный BFS выполняется за 89681 миллисекунд.

Вершины в каждом слое обрабатывались параллельно, с использованием одного из трёх возможных циклов:
* `NonRange : pasl::pctl::parallel_for` без оценки времени обработки каждой вершины (то есть числа инцидентных ей рёбер)
* `NonRangeCost : pasl::pctl::parallel_for` с оценкой времени обработки каждой вершины (то есть числа инцидентных ей рёбер)
* `Range: pasl::pctl::range::parallel_for` с оценкой времени обработки множества вершин (то есть числа рёбер, инцидентных вершинам из этого множества)

Обработка рёбер, инцидентных одной вершине, производилась либо последовательно, либо параллельно.
